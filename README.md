
## Build Setup

**Requires Node.js 22+**

```bash
# install dependencies
npm install

# serve in dev mode, with hot reload
npm run dev

# build for production
npm run generate

# serve in production mode
npm run  preview

```

## Deploy to production
```
npm run  deploy
```
